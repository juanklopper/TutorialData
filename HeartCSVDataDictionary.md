# DATA DICTIONARY FOR THE `"heart.csv"` DATASET

This document provides a detailed description of the columns present in the "heart.csv" dataset, which contains information related to heart disease diagnosis. The "Alternatives" column below shows the words and terms to use when creating plots and visualizations based on this dataset instead of using the column headers which are abbreviated words or words with underscores.

| Column Name       | Data Type      | Description                                                       | Encoding                                                                                              | Alternatives                                                                                  |
|:------------------|:---------------|:------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------|
| Age               | Integer        | Age of the patient in years                                       |                                                                                                       |                                                                                               |
| Sex               | Categorical    | Sex of subject                                                    | 'M' for male and 'F' for female                                                                       |                                                                                               |
| ChestPainType     | Categorical    | Type of chest pain experienced                                    | 'TA' is Typical Angina, 'ATA' is Atypical Angina, 'NAP' is Non-Anginal Pain, 'ASY' isAsymptomatic     | Use "Chest pain type" instead of "ChestPainType" and the information in the "encoding column" | 
| RestingBP         | Integer        | Resting blood pressure (in mm Hg) on admission to the hospital    | 0 values indicate missing data                                                                        | Use "Resting blood pressure" instead of "RestingBP"                                           |
| Cholesterol       | Integer        | Serum cholesterol in mg/dL                                        | 0 values indicate missing data                                                                        |                                                                                               |
| FastingBS         | Binary         | Indicates if fasting blood sugar is greater than 120 mg/dL        | 0 = False, 1 = True                                                                                   | Use "Fasting blood sugar" or "Fasting blood sugar level" instead of "FastingBS"               |
| RestingECG        | Categorical    | Resting electrocardiographic (ECG) results                        | 'Normal', 'ST' (ST-T wave abnormality), 'LVH' (Left ventricular hypertrophy)                          | Use "Resting electrocardiogram" or "ECG" instead of "RestingECG"                              |
| MaxHR             | Integer        | Maximum heart rate achieved in beats per minute                   |                                                                                                       | Use "Maximum heart rate" or "Max HR" instead of "MaxHR"                                       |
| ExerciseAngina    | Binary         | Exercise-induced angina                                           | 0 = No, 1 = Yes                                                                                       | Use "Exercise-induced angina" or "Angina during exercise" instead of "ExerciseAngina"         |
| Oldpeak           | Float          | ST depression induced by exercise relative to rest measured in mm |                                                                                                       | Use "ST depression" instead of "Oldpeak"                                                      |
| ST_Slope          | Categorical    | Slope of the peak exercise ST segment                             | 'Up' for Up slope, 'Flat' for Flat slope, 'Down' for Down slope                                       | Use "ST segment slope" instead of "ST_Slope"                                                  |
| HeartDisease      | Binary         | Diagnosis of heart disease                                        | 0 = No heart disease, 1 = Presence of heart disease                                                   | Use "Heart disease diagnosis" or "Heart disease" instead of "HeartDisease"                    |
